# 6. Implementación

## Entorno de desarrollo
- Windows 10
- Node.js
- Python 3

## Servidores
- Frontend: Angular (localhost:4200)
- Backend: FastAPI (localhost:8000)

## Base de datos
- MySQL

## Pasos de ejecución

1. Ejecutar backend:
   uvicorn backend.main:app --reload

2. Ejecutar frontend:
   ng serve

3. Acceder a:
   http://localhost:4200
