# 2. Arquitectura del Sistema

El sistema sigue una arquitectura cliente-servidor.

## Frontend
- Angular
- HTML, CSS
- Comunicación con API REST

## Backend
- FastAPI (Python)
- SQLAlchemy (ORM)
- Uvicorn

## Base de Datos
- MySQL / MariaDB

## Comunicación
- HTTP (REST API)
- JSON

## Patrón utilizado
- MVC (Modelo-Vista-Controlador)
- Uso de componentes (Angular)
